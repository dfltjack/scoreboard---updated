Alterações personalizadas nas regras

A seguir, você pode adaptar a lógica do placar diretamente modificando a configuração JSON abaixo. Os tempos estão em milissegundos. Muitas configurações (por exemplo, "osaekomi_wazari_time", "osaekomi_ippon_time", "osaekomi_warn_unassigned", "count_wazaris_towards_ippon") podem ser definidas como "null" (sem as aspas) para desativá-las.

Adapte o JSON e clique em "Atualizar e Redefinir" para aplicar as alterações. Para usar este placar em um torneio, proceda da seguinte forma:
  • Prepare um laptop e conecte-o a um segundo monitor (ou projetor).
  • Baixe o código-fonte da página da web scoreboard.zip de a última versão no GitHub.
  • Extraia o zip para qualquer diretório.
  • Abra o arquivo extraído index.html.
  • Abra uma nova aba somente de visualização clicando no ícone de olho.
  • Mova a aba aberta para o segundo monitor.
  • Clique na aba movida e pressione F11 para deixar a página em tela cheia.
  • Mova o mouse de volta para a aba original e use o placar a partir daí. A maioria dos botões também pode ser acionada pelo teclado atalhos, que são anotados como informações de hover nos respectivos botões. Recomendamos rotular o teclado usando adesivos, como neste exemplo.

Para adaptar o tamanho da fonte (para telas menores/maiores), basta aumentar ou diminuir o zoom (Google Chrome/Firefox: Ctrl+Roda do Mouse). Se você quiser aumentar o zoom na página somente de visualização separadamente, reconfigure o Firefox da seguinte forma:

  • Digite "about:config" na barra de endereços e pressione enter.
  • Se o Firefox perguntar, diga que você terá cuidado.
  • Procure a string "browser.zoom.siteSpecific". Se não existir, crie uma, é uma string booleana.
  • Defina como falso (Clique com o botão direito > "Alternar" ou clique em "Alternar").
